Privacy

개인정보처리방침

SolForge는 현재 회원가입, 로그인, 결제, 댓글, 파일 업로드 기능을 제공하지 않는 정적 웹 도구입니다. 사용자가 도구에 입력하는 SQL 쿼리와 EXPLAIN 결과는 브라우저 내 JavaScript로 처리되며, SolForge 자체 서버로 저장하거나 전송하지 않습니다.

처리하는 정보

현재 사이트 기능은 사용자가 직접 입력한 텍스트를 브라우저 메모리에서 가공합니다. 페이지를 새로고침하거나 브라우저 탭을 닫으면 입력값은 사이트가 별도로 보관하지 않습니다. 다만 브라우저 확장 프로그램, 사용자의 클립보드, 기기 환경은 SolForge가 통제하지 않습니다.

광고와 분석 도구

향후 Google AdSense 또는 웹 분석 도구가 적용될 수 있습니다. 이 경우 Google 등 제3자 서비스가 쿠키, 광고 식별자, 접속 로그, 브라우저 정보, 대략적인 위치 정보를 처리할 수 있으며, 관련 정책과 동의 안내는 실제 도입 시점에 갱신합니다.

이용자 유의사항

운영 DB의 비밀번호, API 토큰, 고객 개인정보, 영업비밀, 내부망 주소가 포함된 쿼리는 공개 또는 공유 환경에서 붙여넣지 않는 것을 권장합니다. SolForge는 도구 사용 편의를 제공하지만 입력 내용의 보안성 판단은 사용자가 직접 확인해야 합니다.

정책 변경

기능 추가, 광고 적용, 문의 채널 공개 등 운영 방식이 바뀌면 이 문서를 업데이트합니다. 최종 수정일은 2026년 6월 11일입니다.
