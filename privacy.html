개인정보처리방침

SolForge는 회원가입, 로그인, 결제와 댓글 기능을 제공하지 않는 정적 웹 도구입니다. 사용자가 도구에 입력하는 텍스트, 날짜, 쿼리와 선택한 로컬 파일은 원칙적으로 브라우저 안에서 처리되며 SolForge 자체 서버에 저장하지 않습니다. 다만 사이트 운영, 문의 응대, 광고와 제휴 배너 표시 과정에서 아래 정보가 처리될 수 있습니다.

1. 처리하는 정보

도구 입력값: SQL, JSON, 텍스트, 날짜, 숫자, 로컬 파일 등 사용자가 각 도구에 입력한 값은 브라우저 메모리에서 처리됩니다. 페이지를 새로고침하거나 탭을 닫으면 SolForge가 이를 별도 보관하지 않습니다.

접속 정보: Cloudflare 등 호스팅 인프라는 보안, 캐시, 장애 대응을 위해 IP 주소, 요청 URL, 브라우저 정보, 접속 시간, 대략적인 위치 정보와 같은 로그를 처리할 수 있습니다.

문의 정보: 이메일 문의를 보내는 경우 발신 이메일 주소, 문의 내용, 첨부 파일, 발송·수신 시간이 이메일 제공자와 SolForge 문의 응대 과정에서 처리될 수 있습니다.

2. 처리 목적

SolForge는 도구 기능 제공, 오류 확인, 보안과 악용 방지, 문의 응대, 서비스 개선, 광고와 제휴 운영, 법적 의무 이행을 위해 필요한 범위에서 정보를 처리합니다.

3. 브라우저 권한

음성 인식, 장치 진단, 카메라·마이크·클립보드·파일 선택 등 일부 기능은 브라우저 권한을 요청할 수 있습니다. 권한은 사용자가 해당 기능을 직접 실행할 때 요청되며, 브라우저 설정에서 언제든 철회할 수 있습니다. 브라우저와 운영체제 제공자의 처리 방식은 각 제공자의 정책을 따릅니다.

4. 광고와 쿠키

SolForge에는 Google AdSense 광고가 표시될 수 있습니다. Google을 포함한 제3자 광고 사업자는 사용자의 SolForge 방문 또는 다른 웹사이트 방문 기록을 바탕으로 광고를 게재하기 위해 쿠키를 사용할 수 있습니다. Google은 광고 쿠키를 통해 SolForge와 인터넷상의 다른 사이트 방문 정보를 바탕으로 맞춤 광고를 제공할 수 있습니다.

사용자는 Google 광고 설정에서 맞춤 광고를 관리하거나 거부할 수 있으며, Google의 개인정보 처리 방식은 Google 개인정보처리방침에서 확인할 수 있습니다. 브라우저 설정에서 쿠키 저장을 제한할 수도 있지만, 이 경우 일부 광고 또는 사이트 기능이 예상과 다르게 동작할 수 있습니다.

5. 쿠팡 파트너스 제휴 고지

SolForge에는 쿠팡 파트너스 배너 또는 제휴 링크가 포함될 수 있습니다. 해당 링크를 통해 상품을 구매하면 SolForge는 쿠팡 파트너스 활동의 일환으로 이에 따른 일정액의 수수료를 제공받습니다. 제휴 배너와 링크는 광고 또는 제휴 안내와 가까운 위치에 표시해 이용자가 광고임을 알 수 있도록 운영합니다.

쿠팡 파트너스 바로가기 아이콘이나 제휴 링크를 운영할 때 수신자의 사전 동의 없이 메신저, SNS 등으로 광고성 메시지를 발송하지 않습니다.

6. 외부 서비스와 공개 API

일부 도구는 npm Registry, jsDelivr, Nager.Date 같은 공개 API 또는 브라우저 내장 기능을 사용할 수 있습니다. 해당 기능을 실행하면 사용자의 요청 정보가 외부 서비스로 전송될 수 있으며, 각 서비스의 정책과 이용 조건이 적용됩니다.

7. 보관과 파기

도구 입력값과 로컬 파일은 SolForge 자체 서버에 저장하지 않습니다. 이메일 문의는 문의 처리와 분쟁 대응에 필요한 기간 동안 보관될 수 있으며, 더 이상 필요하지 않으면 삭제합니다. 호스팅·광고·이메일 제공자가 처리하는 로그와 쿠키는 각 제공자의 보관 기준을 따릅니다.

8. 이용자의 권리

이용자는 브라우저 설정을 통해 쿠키를 삭제하거나 차단할 수 있고, 브라우저 권한을 철회할 수 있습니다. 이메일 문의와 관련한 열람, 정정, 삭제 요청은 문의 페이지의 안내에 따라 요청할 수 있습니다.

9. 이용자 유의사항

운영 DB의 비밀번호, API 토큰, 고객 개인정보, 영업비밀, 내부망 주소가 포함된 쿼리는 공개 또는 공유 환경에서 붙여넣지 않는 것을 권장합니다. SolForge는 도구 사용 편의를 제공하지만 입력 내용의 보안성 판단은 사용자가 직접 확인해야 합니다.

10. 정책 변경

기능 추가, 광고 적용, 문의 채널 공개 등 운영 방식이 바뀌면 이 문서를 업데이트합니다. 최종 수정일은 2026년 6월 18일입니다.
