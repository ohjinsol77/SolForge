Free MySQL Toolkit

쿼리를 정리하고 실행 계획을 읽는 시간을 줄입니다.

SolForge는 MySQL 개발자가 매일 반복하는 쿼리 정렬과 EXPLAIN 결과 해석을 브라우저에서 빠르게 처리하도록 만든 정적 웹 도구입니다. 초기 화면에는 필요한 안내만 두고, 실제 기능은 각 도구 페이지에서 독립적으로 사용할 수 있게 분리했습니다.
