기능 소개

          
반복적인 데이터베이스 점검 시간을 줄이는 도구

          

            

              
쿼리 정렬과 문법 강조

              
SELECT, JOIN, WHERE, GROUP BY, ORDER BY 같은 MySQL 예약어를 구분하고 문자열, 숫자, 함수, 변수, 테이블 식별자를 다른 색으로 표시합니다. 긴 쿼리를 붙여넣으면 줄바꿈과 들여쓰기를 자동으로 적용해 코드 리뷰나 공유 전에 읽기 좋은 형태로 바꿉니다.

            

            

              
EXPLAIN 결과 시각화

              
MySQL Workbench에서 확인하던 실행 계획의 흐름을 웹에서 볼 수 있도록 단계 카드와 표를 함께 제공합니다. mysqlsh 스타일의 표, 일반 ASCII 테이블, 탭으로 구분된 결과, JSON 형식의 EXPLAIN FORMAT=JSON 입력을 최대한 자동 인식합니다.

            

            

              
브라우저 안에서만 처리

              
SolForge의 현재 기능은 서버 API 없이 사용자의 브라우저에서 실행됩니다. 붙여넣은 쿼리와 EXPLAIN 결과는 외부 서버로 전송하지 않으며, 정적 사이트 구조라 Cloudflare Pages 같은 CDN 기반 호스팅에 적합합니다.

            

          

        


        

          
사이트 소개

          
SolForge는 개발자 작업 흐름에 맞춘 무료 웹 도구 모음입니다.

          
이 사이트는 백엔드 개발자, 데이터베이스 관리자, 풀스택 개발자가 MySQL 쿼리를 더 빠르게 검토하도록 돕기 위해 제작되었습니다. 초기 기능은 Query Prettier와 EXPLAIN Visual 두 가지이며, 이후 인덱스 설계 점검, 쿼리 비교, JSON 포맷터, 로그 분석 같은 기능을 같은 메뉴 구조 안에 확장할 수 있도록 구성했습니다.

          
디자인은 밝은 배경, 명확한 좌측 탐색, 도구 카드 중심의 레이아웃을 사용합니다. 기능 영역은 다크모드를 지원하므로 긴 코드와 실행 계획을 볼 때 눈의 피로를 줄일 수 있습니다. 모든 주요 텍스트는 검색엔진이 이해할 수 있도록 실제 HTML 문서 안에 배치되어 있으며, 개발자 도구라는 주제를 설명하는 소개와 기능 설명을 포함합니다.

        


        

          
개인정보처리방침

          
입력 데이터는 브라우저에서 처리됩니다.

          
SolForge는 현재 별도의 회원가입, 로그인, 결제, 댓글, 서버 저장 기능을 제공하지 않습니다. MySQL 쿼리와 EXPLAIN 결과는 JavaScript를 통해 사용자의 브라우저에서 처리되며, 이 정적 사이트 자체가 해당 입력값을 서버에 저장하거나 전송하지 않습니다.

          
향후 Google AdSense, 웹 분석, 오류 모니터링 같은 외부 서비스가 추가될 경우 해당 서비스 제공자가 쿠키, 광고 식별자, 접속 로그, 브라우저 정보 등을 처리할 수 있습니다. 광고 및 분석 기능을 적용할 때는 이 방침을 업데이트하고, 관련 법령과 플랫폼 정책에 맞는 고지와 선택권을 제공할 예정입니다.

          
사용자는 민감한 개인정보, 비밀번호, 토큰, 고객 데이터, 영업비밀이 포함된 쿼리를 공개 장소에서 붙여넣지 않는 것이 좋습니다. 문의나 정책 수정 요청은 사이트 운영자가 공개하는 연락 수단을 통해 접수할 수 있습니다.
