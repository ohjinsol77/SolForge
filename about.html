About

SolForge 사이트소개

SolForge는 개발자와 일반 사용자가 반복적으로 찾는 작은 작업을 빠르게 처리하도록 만든 정적 웹 도구 사이트입니다. 설치가 필요한 프로그램 대신 개발, 텍스트, 파일, 이미지와 생활 계산 기능을 브라우저에서 바로 제공하는 것이 목표입니다.

운영 목적

SQL과 실행 계획 분석뿐 아니라 텍스트 정리, 데이터 변환, 로컬 파일 처리와 생활 계산처럼 자주 반복되는 작업을 하나의 일관된 화면에서 제공하는 것을 목표로 합니다.

기술 구성

SolForge는 HTML, CSS, JavaScript로 구성된 정적 사이트입니다. 현재 기능은 서버 API나 데이터베이스 없이 사용자의 브라우저에서만 실행되며, Cloudflare Pages 또는 Workers Assets 같은 정적 호스팅 환경에 맞춰 배포됩니다.

광고와 제휴 운영

SolForge는 무료 도구 운영을 위해 Google AdSense 광고와 쿠팡 파트너스 배너를 사용할 수 있습니다. 광고와 제휴 링크는 콘텐츠와 구분되는 위치에 배치하고, 쿠팡 파트너스 링크를 통한 구매에 대해 일정액의 수수료를 제공받는다는 사실을 이용자가 알아볼 수 있도록 표시합니다.

주의 사항

EXPLAIN Visual의 비용 값은 MySQL이 복사 결과에 직접 제공한 값이 없는 경우 rows, filtered, access type, Extra 정보를 기반으로 한 추정치입니다. 실제 운영 쿼리 튜닝은 MySQL 서버의 통계, 인덱스 상태, 데이터 분포, EXPLAIN ANALYZE 결과와 함께 판단해야 합니다.
