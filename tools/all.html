작업에 맞는 도구를 고르고 결과를 확인하는 방법

SolForge 도구는 빠른 확인과 반복 작업을 줄이는 데 초점을 맞춥니다. 검색과 카테고리로 후보를 좁힌 뒤 각 도구의 입력 기준과 결과 한계를 확인하세요.

이름보다 작업 목적부터 정하세요

개발자 도구는 JSON·SQL·인코딩·패키지 확인, 텍스트 도구는 글자 수·공백·중복 행 정리, 이미지·미디어 도구는 파일 변환과 미리보기에 맞춰 분류했습니다. 검색어가 정확히 떠오르지 않으면 카테고리를 먼저 선택하고 입력 형식이 맞는 도구를 비교해 보세요.

브라우저 처리와 외부 조회를 구분하세요

대부분의 변환·계산은 현재 브라우저에서 실행되며 입력값을 SolForge 서버에 저장하지 않습니다. 시세, 공휴일, npm 정보처럼 최신 공개 데이터가 필요한 도구는 화면에 표시된 제공처 API를 조회하므로 비밀키, 내부 주소, 개인정보를 검색어에 넣지 마세요.

결과의 기준과 단위를 다시 확인하세요

날짜 포함 여부, 시간대, 반올림, 문자 인코딩, 파일 형식에 따라 같은 입력도 다른 결과가 나올 수 있습니다. 계약·투자·의료·행정처럼 실제 결정에 쓰는 값은 도구 설명에서 계산 기준을 확인하고 공식 자료나 전문 서비스로 최종 검증하세요.

원본을 보관하고 작은 예제로 먼저 시험하세요

텍스트와 파일을 변환할 때는 원본을 별도로 보관하고, 일부 데이터로 결과 형식과 손실 여부를 먼저 확인하는 편이 안전합니다. 원하는 결과가 나오면 전체 자료에 적용하고 다운로드 파일·복사된 문자열·계산 기준일을 한 번 더 점검하세요.
