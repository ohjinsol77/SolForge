Tool 01

MySQL Query Prettier

복잡한 MySQL 쿼리를 실제 리뷰 가능한 형태로 줄바꿈, 들여쓰기, 문법 강조합니다.

Formatted output

            

          

        

        

          

            
정리 대상 쿼리
SELECT, JOIN, WHERE, GROUP BY, ORDER BY, LIMIT, CTE, CASE, 서브쿼리가 섞인 MySQL 문장을 줄바꿈과 들여쓰기로 정리합니다. 로그에서 복사한 긴 한 줄 쿼리나 코드 리뷰 전에 읽기 어려운 SQL을 빠르게 확인하는 용도에 맞췄습니다.

            
리뷰에 유용한 점
키워드와 절 단위로 구조가 나뉘면 조인 조건 누락, WHERE 조건 위치, 정렬과 집계 기준, LIMIT 적용 범위를 눈으로 확인하기 쉬워집니다. 출력은 그대로 복사할 수 있어 이슈, PR, 장애 분석 메모에 붙여넣기 좋습니다.

            
브라우저 안에서 처리
입력한 SQL은 현재 브라우저에서만 포맷됩니다. 운영 테이블명이나 조건값이 들어간 쿼리를 외부 서버로 보내지 않으며, 정규화나 실행은 하지 않고 사람이 읽기 좋은 형태로만 변환합니다.
