보스 패턴 타이머를 PIP 창으로 띄워두세요.

혼테일 유혹, 갈무리, 공무와 자쿰·피아누스 공무처럼 반복 확인이 필요한 시간을 슬롯별로 관리합니다. 지원 브라우저에서는 작은 PIP 창을 항상 위에 띄워둘 수 있습니다.

아이콘 선택

PIP 지원 범위

이 도구는 Document Picture-in-Picture API를 사용합니다. 최신 Chrome과 Edge 같은 Chromium 계열 PC 브라우저에서 가장 안정적으로 동작하며, 미지원 브라우저에서는 기본 화면 안에서 타이머를 사용할 수 있습니다.

저장 방식

슬롯, 타이머 이름, 남은 시간과 알림 설정은 현재 브라우저의 localStorage에 저장됩니다. 서버로 전송하지 않으며, 브라우저 데이터를 삭제하면 함께 사라집니다.

사용 팁

보스 입장 전 슬롯을 하나 정해두고 필요한 패턴 타이머를 미리 추가하세요. PIP 창을 띄운 뒤 각 타이머의 시작 버튼을 누르면 게임 화면 위에 남은 시간을 작게 유지할 수 있습니다.
