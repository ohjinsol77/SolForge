PIP 작동 방식

각 도구의 PIP 띄우기 버튼을 누르면 브라우저가 별도의 작은 Picture-in-Picture 창을 엽니다. 이 창은 다른 앱 위에 유지되며, SolForge는 선택한 도구의 핵심 상태와 조작 버튼만 다시 그립니다.

PIP 지원 범위

Document Picture-in-Picture API를 지원하는 최신 Chromium 계열 PC 브라우저에서 PIP 창을 사용할 수 있습니다. 미지원 브라우저에서는 페이지 안에서 동일한 도구를 사용할 수 있습니다.

로컬 처리

메모와 타이머 설정은 현재 브라우저에 저장되고, 이미지 파일은 서버로 업로드하지 않고 Canvas에서만 처리합니다.

작은 창 인터페이스

PIP 창은 공간이 작기 때문에 광고나 사이트 내비게이션을 넣지 않습니다. 시계는 시간과 날짜, 타이머는 남은 시간과 진행률, 색상 도구는 색상 값, 메모는 현재 노트처럼 즉시 봐야 하는 정보만 표시합니다.
