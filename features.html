Features

SolForge 기능소개

SolForge는 개발 작업, 텍스트와 파일 처리, 이미지·음성 기능, 일상 계산을 빠르게 돕는 정적 웹 도구 모음입니다. 입력값과 로컬 파일은 별도 업로드 서버를 거치지 않고 브라우저 안에서 처리됩니다.

MySQL Query Prettier

예약어, 함수, 문자열, 숫자, 변수, 테이블명을 구분해 강조하고 SELECT 목록, WHERE 조건, JOIN 조건, GROUP BY, ORDER BY, CTE와 서브쿼리를 줄 단위로 정리합니다. 복사 버튼을 통해 정리된 결과를 바로 가져갈 수 있습니다.

MySQL EXPLAIN Visual

탭으로 구분된 EXPLAIN 결과, mysqlsh 또는 CLI에서 복사한 ASCII 표, JSON 형식의 실행 계획을 분석합니다. 테이블 접근 순서, 조인 흐름, 인덱스 사용 여부, rows와 filtered 기반의 예상 비용을 카드와 표로 표시합니다.

다크모드와 도구별 페이지

코드와 실행 계획을 오래 보는 사용자를 위해 다크모드를 지원합니다. 각 기능은 별도 URL에서 독립적으로 동작하므로 즐겨찾기, 공유, 추후 기능 확장이 쉽습니다.

텍스트와 개발자 도구

글자수, 텍스트 정리, 한영타, 개인정보 마스킹, 숫자·한글 금액, JSON, Base64 텍스트·파일 변환, URL, Timestamp, UUID, chmod와 비밀번호 생성 기능을 제공합니다.

확장 도구 열기

파일과 미디어 도구

특수문자와 코드표, 색상, 텍스트 비교, 자막 변환, HTML 미리보기, 파일 체크섬, 이미지 압축과 Data URL, TTS·STT, EML과 키보드 이벤트를 지원합니다.

파일·미디어 도구 열기

생활 계산기

나이·만나이·성년 여부와 살아온 날·다음 생일, 연도별 나이표, 띠·띠동갑·띠궁합·삼재, 날짜 정보와 D-Day, 평일·주말 날짜 목록, 날짜 더하기·빼기, 기념일과 아기 100일 계산을 제공합니다.

생활 계산기 열기

고급 로컬 도구

시간·근무시간 계산, 코드 정렬·압축, Hash·HMAC·PBKDF2·AES-GCM, QR·Code 39, ASCII 아트, GIF 생성·프레임 추출, 매직아이, 숫자야구와 지원 기기의 바코드 스캔·손전등을 제공합니다.

고급 도구 열기

양음력과 달력

한국 음력 기준의 양력·음력 상호 변환, 음력 기념일의 연도별 양력 날짜, 고정 및 음력 공휴일, 손없는 날을 브라우저 안에서 계산합니다.

학교와 재미 도구

출생연도에 따른 초·중·고 입학 및 졸업 예상 연도와 학년별 학생 나이표를 보여주며, 생년월일을 조합한 오락용 이름짓기도 제공합니다.

광고 운영 안내

사이트에는 Google AdSense 운영을 위한 게시자 코드와 개인정보처리방침이 적용되어 있습니다. 광고는 콘텐츠와 구분하고 주요 설명, 입력 영역과 결과를 가리지 않는 위치에서만 표시하며 광고 쿠키와 맞춤 광고 관리 방법은 개인정보처리방침에서 확인할 수 있습니다.

GUIDES

도구를 더 정확하게 쓰기 위한 안내

도구를 바로 실행할 수 있도록 화면은 간결하게 유지하고, 테스트 순서와 결과 해석이 필요한 내용은 별도 가이드에서 설명합니다.

모니터·입력 장치·성능 점검 가이드

화면, 마우스, 키보드, 오디오·카메라·센서와 성능 테스트를 어떤 순서로 보고 해석할지 안내합니다.

로컬 파일·미디어 처리 가이드

이미지, 자막, EML, 체크섬과 HTML 미리보기 도구의 처리 범위와 주의사항을 설명합니다.

한국 생활 계산 기준 안내

나이, 날짜, 음력, 공휴일과 학교 일정 계산의 기준과 예외를 정리했습니다.

PIP·메이플랜드 타이머 활용 안내

지원 브라우저, 브라우저 저장 범위와 타이머 프리셋 사용 시 확인할 사항을 안내합니다.
