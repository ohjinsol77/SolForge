Features

SolForge 기능소개

SolForge는 개발 작업, 텍스트와 파일 처리, 이미지·음성 기능, 일상 계산을 빠르게 돕는 정적 웹 도구 모음입니다. 입력값과 로컬 파일은 별도 업로드 서버를 거치지 않고 브라우저 안에서 처리됩니다.

MySQL Query Prettier

예약어, 함수, 문자열, 숫자, 변수, 테이블명을 구분해 강조하고 SELECT 목록, WHERE 조건, JOIN 조건, GROUP BY, ORDER BY, CTE와 서브쿼리를 줄 단위로 정리합니다. 복사 버튼을 통해 정리된 결과를 바로 가져갈 수 있습니다.

MySQL EXPLAIN Visual

탭으로 구분된 EXPLAIN 결과, mysqlsh 또는 CLI에서 복사한 ASCII 표, JSON 형식의 실행 계획을 분석합니다. 테이블 접근 순서, 조인 흐름, 인덱스 사용 여부, rows와 filtered 기반의 예상 비용을 카드와 표로 표시합니다.

다크모드와 도구별 페이지

코드와 실행 계획을 오래 보는 사용자를 위해 다크모드를 지원합니다. 각 기능은 별도 URL에서 독립적으로 동작하므로 즐겨찾기, 공유, 추후 기능 확장이 쉽습니다.

텍스트와 개발자 도구

글자수, 텍스트 정리, 한영타, 개인정보 마스킹, 숫자·한글 금액, JSON, Base64 텍스트·파일 변환, URL, Timestamp, UUID, chmod와 비밀번호 생성 기능을 제공합니다.
