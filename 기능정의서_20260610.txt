개발자들을 위한 웹 사이트를 제작하려고 해 .
클라우드 플레어의 pages에 올릴거라 html + js로 이루어진 정적인 서비스여야해.
사이트의 테마나 전체적인 이미지는 https://www.xbitlabs.com/ko/ 이 사이트를 참고해서 비슷한 느낌으로 만들어줘 대신 좀더 밝게 그리고 내부 기능창은 다크모드 설정이 가능하도록 해주고 실제 다크모드도 구현해줘
왼쪽에 메뉴 탐색 할 수 있는게 있어야하고 추후에 다른 기능들도 추가할거니까 유지보수가 편리한 구조로 해주고
이 사이트는 추후 구글의 애드샌스 심사를 진행 할 예정임으로 개인정보처리방침, 사이트소개,기능 소개에 대한 문구가 충분히 들어가게 해주고 
SEO최적화도 함께 진행해서 제작해줘
내가 처음으로 원하는 사이트 기능메뉴는 아래와같아 모든 메뉴는 사이트가 다크모드로 변경 가능하다는걸 생각하고 만들어줘
1. mysql query pretter
mysql의 쿼리를 집어넣었을 때 알아서 변수명과 디비명 함수명 같은것과 예약어들을 구분해서 
깔끔한색깔로 보여지게 해줘 또한 들여쓰기를 통해 쿼리를 자동정렬하여보여지게 해줘
텍스트박스를 위에 하나 넣고 쿼리를 사용자가 붙혀넣을 수 있게해줘
위에 텍스트박스에 어떤 글자가 들어오던 자동으로 아래 텍스트박스에 정리할 수 있게 해주고
아래 텍스트박스는 깔끔하게 정렬 된 결과물이 보여지게 해줬으면 해 
입력 텍스트박스와 출력 텍스트박스의 비율은 2:8로 출력물이 좀 더 잘보이게 해줘 
또한 아래 텍스트박스에는 카피 버튼을 오버레이로 띄워서 사용자가 쉽게 결과물을 복사해 갈 수 있도록해줘
카피버튼을 눌렀을 땐 copy is ok ! 라는 불투명한 팝업이 뜨게해줘

2. mysql explain visual
mysql workbench에서 잘 사용했던 내용인데 이걸 웹에서 해줬으면 해 
사용자가 mysql query explain 결과물을 넣으면 자동적으로 workbenth에서 보여주는거처럼
쿼리 분석의 결과를 비쥬얼적으로 볼 수 있게 해줘 
어떤 쿼리 순서로 진행되며 어떤 명령어들을 사용했는지.
mysql 8 버전 이상의 쿼리에 모두 대응하도록 잘 준비해줘
mysqlsh에서 사용하는거처럼 표 형태의 값이 나올 수도 있고 
기타 툴에서 explain결과를 복사해서 붙혀넣어 탭형식의 구분자로도 나올 수 있는 점을 모두 파악해서 인식해줘

이상 두 가지 기능을 일단 만들어줬으면 해
다시한번 말하지만 html+js로 된 정적인 페이지여야만해 
또한 구글 애드샌스가 들어갈 수 있게 기본적인 구성들이 빠지지 않게 해줘 
또한 사이트의 전체적인 테마와 어울어질 수 있도록해줘 

모든 작업이 완료되면 깃에 커밋과 푸시를 해서 pages에서 알아서 빌드될 수 있도록 
오류가 없는지 잘 체크하고 로직이 잘 작동하는지 확인까지 마쳐줘
