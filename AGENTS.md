# SolForge 작업 지침

- 이 사이트는 한국어와 영어를 함께 지원한다. 새 기능, 문구, 화면, 상태 메시지, SEO 메타데이터를 추가하거나 수정할 때는 두 언어 모두 반영한다.
- 원본 HTML/JS/CSS는 최대한 1벌로 유지하고, 언어별 HTML을 수동 복사하지 않는다.
- 정적 배포물은 `npm run build`로 생성되는 `dist/ko/...`, `dist/en/...`, `dist/assets/...`를 기준으로 한다.
- 화면 텍스트는 `data-i18n` 또는 `data-i18n-attrs` 키를 사용하고, 번역은 `src/locales/ko.json` 및 `src/locales/en.json`에 반영한다.
- 한국어 기본 문구는 기존 사이트 문맥을 유지하고, 영어는 자연스러운 표현으로 작성한다. `SolForge`, `SQL`, `npm`, `API` 같은 고유명사와 기술 용어는 억지로 번역하지 않는다.
- 언어 전환 버튼은 현재 페이지 경로를 유지해야 하며, 한국어 화면에서는 `🇺🇸`, 영어 화면에서는 `🇰🇷`이 보여야 한다.
- 변경 후 최소한 `npm run build`와 `npm run check`를 실행해 검증한다.
