HARDWARE CHECKS

화면과 입력 장치를 점검하는 순서

브라우저 테스트는 빠른 이상 확인에 유용하지만 장비 인증이나 수리 진단을 대신하지 않습니다. 한 가지 수치보다 같은 환경에서 반복한 결과와 실제 사용 증상을 함께 보세요.

4점검 단계
LOCAL브라우저 측정

1. 모니터는 환경을 고정하고 확인하세요

밝기 자동 조절, 야간 모드, HDR, 색상 보정 앱을 잠시 끄고 화면을 충분히 예열한 뒤 단색·블랙·화이트·감마·균일도 순서로 확인하세요. 불량화소나 빛샘은 주변 조명과 화면 밝기에 따라 다르게 보일 수 있어, 스크린샷만으로 패널 문제를 확정할 수는 없습니다.

화면 진단 도구 열기

2. 마우스·키보드는 반복 테스트가 중요합니다

더블클릭, 드래그, 고스팅, 폴링레이트 추정은 동일한 USB 포트와 브라우저에서 여러 번 실행하세요. 브라우저 이벤트 간격은 운영체제, 절전 설정, 백그라운드 탭, 무선 연결 상태의 영향을 받으므로 표시값을 제조사 사양의 정확한 측정값으로 보지 마세요.

입력 장치 테스트 열기

3. 권한이 필요한 장치는 목적을 확인하세요

마이크·카메라·모션 센서는 브라우저 권한을 허용해야 동작합니다. 테스트가 끝나면 브라우저의 사이트 권한에서 다시 철회할 수 있으며, 공유 PC에서는 장치 미리보기나 녹음 데이터를 남기지 않도록 주의하세요. 지원 여부는 브라우저와 기기마다 다릅니다.

오디오·카메라·센서 진단 열기

4. 성능 테스트는 안전한 범위에서 짧게 실행하세요

CPU·GPU·메모리 테스트는 현재 브라우저 탭의 짧은 부하와 렌더링 상태를 보는 참고 도구입니다. 발열, 배터리, 다른 실행 프로그램, 전원 모드에 따라 결과가 달라집니다. 기기가 과열되거나 반응이 느려지면 즉시 중단하고, 장기 안정성 검증은 운영체제·제조사 도구를 사용하세요.
